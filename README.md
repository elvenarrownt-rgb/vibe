# Приложение: заявка на покупку песка

Простое веб-приложение на Node.js + Express для оформления заявки на покупку песка.

Пользователь заполняет форму (контакты, адрес, тип песка, объём, комментарий), а сервер принимает заявку и логирует её в консоль.

## Предварительные требования

- Установлен **Node.js** (вместе с `npm`)

## Установка зависимостей

В терминале, в папке проекта (`vibe`), выполните:

```bash
npm install
```

Будут установлены:

- `better-sqlite3`
- `express`
- `cors`

## Запуск приложения

```bash
node server.js
```

или (после настройки `npm`):

```bash
npm start
```

Сервер запустится по адресу: `http://localhost:3000`

## Использование

1. Откройте в браузере `http://localhost:3000`
2. Заполните форму:
   - ФИО / название компании
   - телефон
   - адрес доставки
   - тип песка
   - объём и единицы измерения
   - (опционально) e-mail, желаемая дата, комментарий
3. Нажмите «Отправить заявку»

При успешной отправке:

- пользователь увидит сообщение об успехе
- данные заявки появятся в консоли Node.js и сохранятся в базе SQLite (`data/orders.db`)
